fkit (fiber-kit) is a fiber section analysis program built with python. Perform moment-curvature and P+M interaction analysis in less than 5 lines of code. Originally meant for reinforced concrete sections, it was later extended to encompass material of all types. Fibers can be defined by the user with a plethora of built-in material models (e.g. Hognestad, Manders, RambergOsgood, Menegotto-Pinto, ElasticPlastic, and more).

I thought the project turned out great. One of my best yet. Beautiful object-oriented design and beautiful visualizations. Link to GitHub repo


EZAnchor is a Python applet that performs nonstructural components seismic force (Fp) calculations per ASCE 7-16 Chapter 13, it then applies Fp in all possible directions (all 360 degrees) to determine the maximum shear and tension in the anchors holding the equipment in place.

The impetus for this package came from a lengthy technical discussion at work. Although the technical background behind equipment anchorage might be simple, there's is actually tons of nuance and this program is far from easy to implement. Read more here in the GitHub repo


fapp, pronounced "F - app", stands for Frame Analysis Program in Python. It is a lightweight and easy-to-use Python implementation of the Finite Element Method (FEM). Perform first-order elastic analyses of any 3-D frame structure and visualize results with a fully interactive web browser interface.

I've implemented the direct stiffness method several times throughout my education. First in Excel/VBA, then in MATLAB, and now in Python. I think I've really struck gold here. The entire source code is only around 500 lines if you don't count comments and the plotting functionalities. Link to GitHub repo

spColumn is a concrete design software commonly used for strain compatibility of concrete columns and walls. One of the best features of the program is that it allows user to create model files in ‚Äú.cti‚Äù format; which are plain text files that can be opened and viewed by any text editor. Furthermore, .cti files can be executed in batches from the command line, allowing for great flexibility in automation. More info from StructurePoint here.

I wrote a python script that extract column/wall info from an input spreadsheet to batch generate and analyze spColumn section as well as export results in .png format. With this tool, you can perform hundreds section analysis in mere seconds. Supports rectangular columns, I-shape, C-shape, L-shape, and T-shape walls.
